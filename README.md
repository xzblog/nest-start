> [Nest](https://github.com/nestjs/nest) 快速开始模板，基于包含一般管理后台的一些常用功能，如：用户管理、角色管理、日志管理等。

## 运行项目

```bash
# install
$ pnpm install

# watch mode
$ pnpm run start:dev

```


## 特征
- [ ] 支持全局响应、异常拦截器，统一响应格式
- [ ] 支持请求参数校验
- [ ] 支持swagger文档
- [ ] 支持接口鉴权 基于JWT
- [ ] 支持prisma操作数据库
- [ ] 支持redis缓存
- [ ] 支持上传文件
- [ ] 支持短信发送
- [ ] 支持excel导出
- [ ] 支持邮件发送
- [ ] 支持定时任务
- [ ] 支持日志记录
